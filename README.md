# visualifter
A weightlifting visualizer: Build and organize workout "blocks".

## Getting Started

- Clone
- Run `bundle install`
- Run `rails db:migrate`
- Run `rails db:seed`
- Run `rails s`
- Go to `index.html` in your browser
